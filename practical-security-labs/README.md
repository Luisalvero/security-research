# Practical Security Labs

Status: Planned.

Walkthroughs of hands-on security labs. The focus is methodology over
results: a writeup that documents an unsuccessful attempt and what was
learned from it is more valuable than one that records only the successful
path.

## Index

No reports published yet.

## Conventions

- The walkthrough leads with the methodology, not the answer.
- At least one wrong path taken and what was learned from it is
  documented.
- The lab and platform are cited; paid content is not pasted verbatim.
- Flags are redacted where the platform requires it.
- Each report records: tools used, time taken, key insight.

## Candidate platforms

- TryHackMe (rooms in the active learning path)
- HackTheBox (retired machines only; active machines violate terms of service)
- PortSwigger Web Security Academy
- pwn.college, OverTheWire
