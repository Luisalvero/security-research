# Reverse Engineering

Status: Planned.

Static and dynamic analysis of binaries, firmware, and protocols. Each
report in `reports/<target>/` will be self-contained: writeup plus
supporting artifacts (Ghidra exports, disassembly excerpts, hex dumps).

## Index

No reports published yet.

## Conventions

- Targets are limited to those with explicit legal authorization to study:
  CTF binaries, sample binaries, the author's own software, or commercial
  software where the EULA permits interoperability research.
- Raw copyrighted binaries are not committed. Notes, screenshots, and
  decompilation excerpts are.
- Each report includes the methodology used, not only the result. Process
  is the point.

## Report template

Reports follow the standard template used across this repository:
overview, target, tools, methodology, technical breakdown, findings,
lessons learned, next steps.
